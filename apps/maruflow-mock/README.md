# MaruFlow MVP Mock

紙テスト採点支援アプリ **MaruFlow** の MVP を、実装前に画面モックとして具体化したプロジェクトです。

## 概要

- **アプリ名**: MaruFlow MVP Mock
- **目的**: 紙テスト採点ワークフローのUI/UXを画面モックで具体化し、MVP仕様を可視化する
- **技術スタック**: React 19 + Vite 8 + TypeScript 6（外部UIライブラリ不使用）
- **データ**: 架空のテスト・架空の答案のみ使用。個人情報・実児童データは一切含まない

## 画面構成

| 画面 | 説明 |
|------|------|
| ホーム | ワークフロー全体像とMVP範囲の説明 |
| PDF登録 | 空白問題PDFのアップロードとテスト情報入力 |
| 採点欄設定 | PDF上の問題ゾーン（座標・配点）の定義 |
| 採点 | 答案への丸・バツ・点数・コメントの付与 |
| 出力確認 | 採点結果サマリーと採点済みPDFのダウンロード |

## 実行方法

```bash
pnpm install   # 初回のみ
pnpm dev       # 開発サーバー起動 → http://localhost:5173
pnpm build     # 本番ビルド
pnpm preview   # ビルド結果のプレビュー
```

## MVP 範囲

このモックが対象とするMVP機能は以下の通りです。

- PDF上に手動で丸・バツ・点数・コメントを置く
- 採点欄（ゾーン）の位置と配点を定義する
- 採点済みPDFとして出力する（モック）

以下は次フェーズ以降の実装予定です。

- AI採点（自動丸付け）
- OCR（手書き文字認識）
- クラス管理・名簿連携
- 成績分析ダッシュボード
