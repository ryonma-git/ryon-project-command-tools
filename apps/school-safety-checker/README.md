# 学校安全初動チェックアプリ (School Safety Checker)

学校で事案（児童所在不明、不審者、けが等）が発生した際の初動対応を補助するWebアプリです。
発生時刻や確認エリアの抜け漏れを防ぎ、事後の振り返り用ログを生成します。

## ⚠️ 重要：免責事項とプライバシー設計

1. **正式なマニュアルではありません**
   - 本アプリは初動確認の補助ツールです。
   - 通報・連絡基準や具体的な対応フローは、必ず**各学校・自治体の正式な危機管理マニュアル**に従ってください。
   - 本アプリは判断を自動化するものではありません（「〇分経過したから通報する」などの自動通知は行いません）。

2. **個人情報を保存しない設計**
   - 児童の氏名、写真、住所など、**個人を特定できる情報は絶対に入力しないでください**。
   - アプリは外部APIやデータベースを使用せず、すべてブラウザ上で動作します（静的Webアプリ）。
   - 入力データのブラウザ保存（localStorage）は**初期状態でOFF**になっています。ONにした場合でも、役割カードの「担当者名」は保存されず、ページを閉じると消去されます。

## 主な機能

- **事案開始・タイマー**: 発生時刻と気づいた時刻を記録し、対応開始からの経過時間を表示します。
- **初動チェックリスト**: 管理職への共有、捜索担当の割り振り、連絡要否の確認など、初動で必要なアクションの抜け漏れを防ぎます。
- **捜索エリア管理**: 校内各エリアの確認状況（未確認・確認中・確認済み）をワンタップで切り替えられます。
- **役割カード**: 全体指揮、記録係、校内捜索などの役割分担を可視化します。
- **出力機能**: 入力内容をもとに「対応ログ（Markdown）」「振り返り用メモ」「ChatGPT事後整理プロンプト」を生成し、クリップボードにコピーできます。

## 使い方

1. **事案開始**: 事案種別を選択し、時刻や状況をメモします。タイマーをスタートします。
2. **チェックリスト**: 対応が進むごとにチェックを入れます。
3. **捜索エリア**: 捜索状況に応じてエリアのステータスを更新します。
4. **出力**: 対応が一段落したら、ログをコピーして正式な報告書作成の参考にします。

## 開発・ビルド方法

本アプリは React + Vite + TypeScript で構築されています。

```bash
# 依存関係のインストール
pnpm install

# 開発サーバーの起動
pnpm dev

# 本番用ビルド（dist/ フォルダが生成されます）
pnpm build
```
